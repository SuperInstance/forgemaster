Algorithm Reference [show]

Backtracking: Assign variables one by one, check constraints after each assignment. If a constraint fails, backtrack and try the next value.
MRV (Minimum Remaining Values): Choose the variable with the fewest legal values left first. Fails fast.
LCV (Least Constraining Value): Choose values that leave the most options for unassigned variables.
Forward Checking: After each assignment, prune incompatible values from unassigned variable domains.
AC-3 (Arc Consistency): Preprocess: remove values that have no supporting value in any neighbor variable's domain.
