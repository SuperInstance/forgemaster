### #1 [P0 · SECURITY] Kill Port 4051 → Wire Fleet Auth Layer

**What's broken:** Port 4051 is serving `/tmp` — open directory traversal. Every other action is undermined while this is live.

**Fix:**
- Oracle1: identify what binds 4051 → kill or firewall (`ufw deny 4051`)
- Replace with FLUX ISA `VAUTH` opcode (0xBF range) — lightweight token check for inter-agent calls
- Fleet dashboard auth header: agents call Oracle1 endpoints with fleet token validated against PLATO "auth-config" tile
