- **Bug 2: Power accumulation** — accumulated absolute power instead of delta
  - Fixed: track `power_delta = predicted_power - base_power`, accumulate against headroom

### Multibeam Sonar Beamforming
- **sonar_beamformer.cu**: Delay-and-sum beamforming with 32-element array
- **bench_beamformer.cu**: Full benchmark with synthetic target generation
- **Results**:
  - 32 elements, 128 beams, 2048 samples: 294 pings/sec (37,684 beams/sec)
  - Target detection: 30° synthetic detected at 29.1° (0.9° error, within beam width)
  - Array scaling: 8→16→32→64 elements: 162K → 92K → 49K → 43K beams/sec
  - Shared memory: 8.2KB per block (128B delays + 8KB accumulator), fits 48KB limit
- **Techniques**: Linear interpolation for fractional delays, Hamming window for sidelobe suppression, atomicAdd on shared float

### Key Technical Insights
- **cudaGraph replay**: Eliminates ~43% of per-cycle overhead by replacing 4 kernel launches with 1 graph launch
- **Constraint scheduler**: Multi-objective scoring with hard feasibility filters successfully distributes heterogeneous workloads
- **Beamformer**: Linear interpolation gives first-order delay accuracy; Hamming window reduces sidelobes from -13dB to -43dB
- **Memory hierarchy**: Shared memory critical on Jetson (68 GB/s vs 192 GB/s bandwidth) — optimize access patterns
