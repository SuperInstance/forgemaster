## Lessons
- `kimi -p` works for Python builds at any memory, OOMs on Rust >13GB

- Kimi code quality is notably higher than Pi — proper types, error handling, docs
- Kimi set pad_token=eos_token correctly for cu126 — my manual version missed this
- Direct tensor ops (no DataLoader) is 3.5x faster on small batches
- `pip index versions` output format varies — always check the first line
