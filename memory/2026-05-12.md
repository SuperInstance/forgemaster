### Papers Written (3)

1. **THE-MANDELBROT-FLEET.md** (19KB) — Three operations are one system:
   - Zoom in: fractal detail (every tile boundary is infinitely complex, amnesia cliff at 10% = Mandelbrot boundary)
   - Zoom out: self-similar consolidation (dream module = deflation, baby fleets at every level)
   - Asynchronous spline: each agent interpolates toward projected destination using partial control points
   - z → z² + c maps to: state(t+1) = adjoint(state(t), fleet_model) + new_information
   - Boundary IS the amnesia cliff — divergent orbits = loss of fleet coherence

2. **THE-GOLDEN-TWIST.md** (14KB) — 4D double rotation unified theory:
   - Single 4D rotation R(2π/φ, 2π/φ²) projects to Penrose, Eisenstein, Mandelbrot, and temporal spline
   - The golden twist is quasiperiodic (never repeats, comes arbitrarily close)
   - α/β = φ gives the golden angle (137.51°) — same angle as sunflower phyllotaxis
   - 5D keel = 4D golden twist + temporal projection axis
   - Icosian ring (quaternions with φ coefficients) is the symmetry group
   - Constitution = icosahedral group action on fleet state space
   - Explains: baton split = 3, crystallization t* ≈ 3-4, temp 1.0 optimal, 28:1 speed ratio, 10% cliff

3. **PENROSE-MEMORY-PALACE.md** (14KB) — Aperiodic coordinates for AI retrieval:
   - Problem: vector DBs have identical neighborhoods everywhere — no location awareness
   - Penrose solution: every finite patch is unique, so retrieval context IS location
   - Three properties: unique neighborhoods, long-range order (Bragg peaks), exactly 3-colorable
   - 3-coloring = baton shards (Built/Thought/Blocked), guaranteed no adjacent echo chambers
   - Bragg peak retrieval: query diffracts against stored tiling, peak intensity = confidence
   - Golden hierarchy: deflation levels at φ^k scale (fact → session → project → domain → fleet)
   - Comparison table vs HNSW: uniqueness, zero collision, structural retrieval, intrinsic location

```
Fleet(t) = Project₂D [ R(2π/φ, 2π/φ²) · IcosianEmbed(Keel(t)) ]

- Penrose = xy-projection

- Spline = Fleet(t) as function of t
- Amnesia = |dFleet/dt| (rate of twist = rate of forgetting)
- Memory palace = Penrose coordinates for stored tiles
