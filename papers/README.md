### EMSOFT 2026: FLUX Constraint Architecture

- [`emsoft-2026-flux.tex`](emsoft-2026-flux.tex) — Initial submission
- [`emsoft-2026-flux-v2.tex`](emsoft-2026-flux-v2.tex) — IEEEtran format, structural fixes

**Abstract:** The FLUX constraint architecture provides mathematically proven safety guarantees for distributed constraint checking on safety-critical systems. Using the Eisenstein integer lattice (A₂) as the computational substrate, the system achieves provably optimal snap-to-lattice operations with bounded error (ρ = 1/√3 ≈ 0.5774) and zero differential drift across 20+ GPU experiments.
