# Forgemaster


FLUX runtime. Probes system, discovers compilers (6 toolchains), compiles kernels across 14 language targets, benchmarks all, picks the fastest. Discovered Python `hash()` at 84ns beats C at 256ns for keys under 64 bytes. 200ms compilation timeout per kernel. 240 Rust, 133 Python, 31 C files.


Apache 2.0 — Cocapn fleet infrastructure.
