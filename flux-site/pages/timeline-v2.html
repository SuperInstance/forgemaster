PLATO → FLUX Lineage Timeline

The 60+ year evolution of constraint-based software from early teaching machines to modern formal compliance tooling

1960

PLATO (Bit-Serial Teaching Machine)

The first computer-assisted instruction system, developed at the University of Illinois. Used bit-serial hardware to minimize memory usage for shared terminal access across multiple campuses.

Connection: Laid foundational groundwork for structured, rule-based educational programming languages.

1970s

TUTOR Authoring Language

PLATO's proprietary programming language, introduced judging blocks — constraint loops that validated student input against predefined logical rules to enable interactive, adaptive courseware.

Connection: Pioneered constraint-based input validation that evolved into formal constraint logic programming paradigms.

1980s

Computer-Aided Instruction (CAI) Scale

PLATO-derived CAI systems expanded to university and corporate training markets, refining modular, rule-based course structures and formalizing best practices for interactive, constraint-driven learning tools.

Connection: Bridged educational tooling to industrial software reliability, paving the way for formal verification practices.

1990s

Constraint Logic Programming (CLP)

Emerging academic field that formalized constraint-based problem solving, building directly on TUTOR's judging blocks and CAI's rule frameworks to enable automated validation of complex logical constraints.

Connection: Connected educational tooling to formal software verification, creating a pathway for industrial adoption.

2000s

Formal Verification (CompCert, seL4)

Industry-wide shift to mathematically proving software correctness, using CLP principles to verify critical systems including the CompCert verified C compiler and seL4 microkernel, the first formally verified OS kernel.

Connection: Established regulatory and technical foundations for safety-critical software compliance standards.

2010s

DO-254/ISO 26262 & Aerospace Standards

Global safety standards for aerospace (DO-178C/DO-254) and automotive (ISO 26262) industries, mandating traceable, verified constraint-based design practices for safety-critical systems.

Connection: Created mandatory compliance pathways for modern constraint compiler tooling.

2024

FLUX Constraint Compiler

Cocapn's flagship tool, a modern constraint compiler that unites PLATO's educational roots, TUTOR's judging blocks, CLP's formal logic, and formal verification standards to generate provably correct, safety-compliant code for aerospace, automotive, and medical industries.

Connection: Full circle of the lineage, delivering industrial-grade constraint tooling from the original PLATO teaching machine.
